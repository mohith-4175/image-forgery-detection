About the Project

A hybrid forensic image analysis system combining Error Level Analysis with Machine Learning and Deep Learning.

📋

System Overview

This project presents an intelligent Image Forgery Detection System that integrates Error Level Analysis (ELA) with both Machine Learning (ML) and Deep Learning (DL) techniques to identify digitally manipulated images.

ELA is first applied to highlight compression inconsistencies that commonly occur in tampered regions. From these ELA-transformed images, handcrafted forensic features are extracted and analyzed using Machine Learning models for initial classification.

To further enhance accuracy and robustness, a Deep Learning approach based on the EfficientNet-B3 architecture is employed, which automatically learns high-level spatial and texture patterns indicative of image manipulation.

System Workflow

01

Image Upload & Preprocessing

User uploads image through web interface. System validates format, size, and performs initial quality checks.

02

Error Level Analysis (ELA)

Resaves image at known quality level and computes difference map to identify compression inconsistencies.

03

Feature Extraction

Extracts statistical features, noise patterns, and texture characteristics from ELA output.

04

Hybrid Classification

Parallel processing through Random Forest (ML) and EfficientNet-B3 (DL) models for ensemble prediction.

05

Explainable AI Generation

Generates attention heatmaps using Grad-CAM to highlight suspicious regions in the image.

06

Result Visualization

Presents verdict with confidence score, visual comparisons, and forensic evidence.
