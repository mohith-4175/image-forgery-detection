System Overview

This project presents an intelligent Image Forgery Detection System that integrates Error Level Analysis (ELA) with both Machine Learning (ML) and Deep Learning (DL) techniques to identify digitally manipulated images. ELA is first applied to highlight compression inconsistencies that commonly occur in tampered regions. From these ELA-transformed images, handcrafted forensic features are extracted and analyzed using Machine Learning models for initial classification.

To further enhance accuracy and robustness, a Deep Learning approach based on the EfficientNet-B3 architecture is employed, which automatically learns high-level spatial and texture patterns indicative of image manipulation. The combined ML and DL framework enables reliable detection of forged and authentic images while also supporting visual interpretability through ELA and heatmap-based analysis, making the system effective for forensic investigation and real-world digital media verification.

System Workflow

1

Image Upload & Preprocessing

User uploads image through web interface. System validates format, size, and performs initial quality checks.

2

Error Level Analysis (ELA)

Resaves image at known quality level and computes difference map to identify compression inconsistencies.

3

Feature Extraction

Extracts statistical features, noise patterns, and texture characteristics from ELA output.

4

Hybrid Classification

Parallel processing through Random Forest (ML) and EfficientNet-B3 (DL) models for ensemble prediction.

5

Explainable AI Generation

Generates attention heatmaps using Grad-CAM to highlight suspicious regions in the image.

6

Result Visualization

Presents verdict with confidence score, visual comparisons, and forensic evidence.
